# 🛍️ Customer Segmentation using K-Means Clustering

## 📝 概要 (Overview)
架空のショッピングモールの顧客データ（年齢、年収、支出スコア）を用いて、K-Meansクラスタリングによる顧客セグメンテーション（教師なし学習）を実装したもの。
単なるアルゴリズムの実装にとどまらず、**「データからビジネス上のインサイトを抽出し、マーケティング戦略に落とし込むこと」**を目的としている。

- **データセット:** Mall Customer Segmentation Data (Kaggle)
- **分析手法:** K-Means Clustering, エルボー法 (Elbow Method)
- **使用技術:** Python, Pandas, Scikit-learn, Matplotlib, Seaborn, Plotly (3D可視化)

---

## 🚀 実践したこと

### 1. 探索的データ分析 (EDA)
- `Seaborn`を用いて、顧客の年齢・年収・支出スコアの分布を可視化。
- 変数間の相関や、性別ごとの購買傾向の差を分析し、クラスタリング前の仮説構築を実施。

### 2. 最適なクラスター数（K）の科学的証明
- K-Meansアルゴリズムを実行する際、主観的な分割を避けるため**エルボー法（Elbow Method）**を実装。
- クラスター内誤差平方和（WCSS）を計算・グラフ化し、曲線の「肘」にあたる部分から、最適なクラスター数が「5」であること示唆出し。

### 3. モデルの構築と3Dインタラクティブ可視化
- `scikit-learn`を用いてK=5でクラスタリングを実行。
- 単純な2Dの散布図ではなく、`Plotly`を活用して「年収」「支出スコア」「年齢」の3軸による**3D散布図**を構築。非技術者にも直感的に伝わる、説得力のあるビジュアライゼーションを作成。

### 4. ビジネス戦略への翻訳（ペルソナ定義）
機械が分類した5つのクラスターに対し、マーケティング施策を見据えたペルソナを設定。
- 🏆 **Target (高年収×高支出):** VIP待遇や新商品のクローズド先行販売を狙う層。
- 🛡️ **Careful (高年収×低支出):** 初回ハードルを下げる強力なインセンティブで来店を促す層。
- 🛍️ **Careless (低年収×高支出):** 手頃な価格帯や分割払いのプロモーションを訴求する層。
- 💼 **Sensible (低年収×低支出):** 大規模セールの時のみアプローチし、コストを抑える層。
- 👥 **Standard (中年収×中支出):** ロイヤリティプログラムで上位クラスターへの引き上げを狙う層。

## 🧠 学んだこと・得られた知見 (What I Learned)

* **「正解のないデータ」に対するアプローチの重要性**
  これまで経験してきた「予測精度（Accuracy）」を競う教師あり学習とは異なり、教師なし学習では**「いかにビジネスとして解釈可能で、アクションに繋がる分類ができるか」**が評価軸になることを学んだ。
* **技術的根拠とビジネス的解釈の橋渡し**
  「なんとなく5つに分けた」ではなく、エルボー法で数学的根拠を示した上で、それぞれのグループに「Target」「Careless」といった実務的なラベル（意味）を与えることで、分析結果の価値が飛躍的に高まる。
