## 📌 概要 (Overview)
メディアビジネスにおけるコンテンツ投資戦略（特に非英語コンテンツへのシフト）をテーマに、**ベイズ線形回帰（Bayesian Linear Regression）**を用いた成長予測パイプラインを構築しました。

メディアプランナーにとって、単一の予測値よりも「その予測のブレ幅（不確実性）」を理解することの方が、数億円規模の予算配分においては重要です。本プロジェクトでは、`PyMC`を使用してトレンドの傾きと不確実性を定量化し、**「不確実性のコーン（Cone of Uncertainty）」**として可視化することで、リスク調整された意思決定フレームワークを提案します。

---

## 📈 分析アプローチと結果

### 1. データセットと変数の定義
TMDB（The Movie Database）データセットを用い、2000年から2021年までのコンテンツ傾向を分析しました。
* **対象**: 2000年〜2021年のリリース作品
* **目的変数**: 年ごとの非英語コンテンツ比率 (Non-English Content Ratio)
* **手法**: Bayesian Linear Regression (成長の傾きと切片を確率分布として推定)

### 2. 「不確実性のコーン」による可視化
単なる回帰直線を描くのではなく、事後分布からサンプリングした数多の回帰直線を重ね合わせることで、未来に進むほど広がっていく「リスクの帯」を可視化しました。

![Bayesian Forecast](Bayesian%20Forecast.png)

**【分析判断とインサイト】**
* **直線のフェーズ**: 現在は成長の真っ只中にあり、線形モデルが現状のトレンドを最もシンプルかつ正確に説明します。
* **不確実性の定量化**: 予測期間（2026年まで）が延びるにつれ、予測の幅（ティール色の帯）が広がっています。これは、長期計画における予測リスクが時間とともに増大することを視覚的に示しています。

## 🧠 5. 今回の学び (Key Learnings)

### ① なぜベイズ線形回帰を選んだのか？（オッカムの剃刀）
複雑なS字カーブ（ロジスティック成長モデル）への誘惑がありましたが、トレンドの変曲点が未到達のデータに対して無理に複雑な仮定を置くことは避けるべきと判断しました。目的が「直近数年のメディア予算編成」である以上、モデルの解釈性とシンプルさを優先し、線形モデルを選択しました。

### ② 不確実性を可視化することのビジネス価値
ステークホルダーは「来年はどうなる？」と聞いてきますが、プロのアナリストは「どの範囲に収まる可能性が高いか」を答えるべきです。この「不確実性のコーン」を見せることで、過剰投資（リスクの取りすぎ）と機会損失（投資不足）の両方を防ぐ、冷静な議論のテーブルを作ることができました。

### ③ 「データが語らないこと」をモデルで補完する
MCMCサンプリングを通じてパラメータの分布を得るプロセスは、単なる回帰分析以上に「データに対する確信度」を教えてくれます。ベイズ統計は、データが少ない時こそ、強力な意思決定支援ツールになると実感しました。

## 📚 【Appendix】ベイズ線形回帰の基本概念
本プロジェクトで用いた手法の備忘録です。

* **ベイズ線形回帰**: パラメータを単一の値ではなく「確率分布（Posterior Distribution）」として推定します。これにより、予測値の「平均」だけでなく「幅（信用区間）」を計算できます。
* **MCMC (NUTS Sampler)**: パラメータの複雑な確率分布を探索する手法です。計算コストはかかりますが、解析的に解けないモデルでも頑健な推論が可能です。
* **不確実性のコーン**: ベイズモデリングによって得られた事後分布を用いて予測を行うと、未来時点での予測値の分散が大きくなるため、このように放射状に予測幅が広がります。これが「未来の予測は難しい」という統計的事実の可視化です。
